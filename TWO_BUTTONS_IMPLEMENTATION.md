# Две кнопки для создания проекта на Dashboard

## Обзор
На странице Dashboard добавлены две кнопки для создания проекта с разными способами отправки данных.

## Реализация

### 1. Форма на сайте (Website Form)
**Расположение:** Левая карточка

**Особенности:**
- Открывает модальное окно с формой на текущем сайте
- Секретный ключ подставляется автоматически из профиля пользователя
- Все данные остаются в рамках системы
- Удобно для быстрого создания проектов

**Визуальное оформление:**
- Иконка: Документ с плюсом (фиолетовая)
- Кнопка: Градиент от фиолетового к синему
- Описание: "Создайте проект через форму на нашем сайте. Ваш секретный ключ подставится автоматически."

**Действие:** Открывает модальное окно `ProjectSubmitModal`

### 2. Анонимная форма в Telegram
**Расположение:** Правая карточка

**Особенности:**
- Открывает Telegram бот в новой вкладке
- Требует ручного ввода секретного ключа
- Анонимная отправка данных через Telegram
- Для пользователей, предпочитающих работать через мессенджер

**Визуальное оформление:**
- Иконка: Логотип Telegram (голубая)
- Кнопка: Градиент от синего к голубому
- Описание: "Отправьте данные через анонимный Telegram бот. Потребуется ввести ваш секретный ключ вручную."

**Действие:** Переход по ссылке `https://t.me/mytestbot_2025_v1_bot`

## Дизайн

### Адаптивность
- **Мобильные устройства:** Карточки располагаются вертикально (1 колонка)
- **Планшеты и десктоп:** Карточки располагаются горизонтально (2 колонки)

### Цветовая схема
- **Форма на сайте:** Фиолетово-синий градиент (purple → blue)
- **Telegram бот:** Сине-голубой градиент (blue → cyan)

### Эффекты
- Hover: Увеличение масштаба (scale-105) и снижение прозрачности
- Transition: Плавная анимация всех изменений
- Shadow: Тень для создания глубины

## Преимущества решения

1. **Выбор для пользователя** - каждый может выбрать удобный способ
2. **Ясность** - понятные описания и иконки
3. **Визуальное разделение** - разные цвета для разных способов
4. **Доступность** - обе опции равнозначно представлены
5. **Гибкость** - легко добавить третий способ при необходимости

## Сравнение способов

| Характеристика | Форма на сайте | Telegram бот |
|----------------|----------------|--------------|
| Секретный ключ | Автоматически | Вручную |
| Где открывается | Модальное окно | Новая вкладка |
| Анонимность | Нет | Да |
| Скорость | Быстрее | Медленнее |
| Удобство | Выше | Ниже |
| Использование | Основной способ | Альтернативный |

## Использование

### Для пользователей
1. Выберите удобный способ создания проекта
2. Нажмите соответствующую кнопку
3. Заполните форму и отправьте данные

### Для разработчиков
Чтобы изменить ссылку на Telegram бот:
```svelte
<a href="https://t.me/YOUR_BOT_NAME" ...>
```

Чтобы добавить третий способ:
```svelte
<div class="grid grid-cols-1 gap-6 md:grid-cols-3">
  <!-- Добавьте третью карточку -->
</div>
```
