## Рекомендации по внедрению

### 1. Для страниц с существующими тегами
Замените существующие `<svelte:head>` блоки на новые теги из этого документа.

### 2. Для страниц без тегов
Добавьте блок `<svelte:head>` в начало компонента:

```svelte
<svelte:head>
	<title>Заголовок страницы</title>
	<meta name="description" content="Описание страницы" />
	<meta name="keywords" content="ключевые слова" />
</svelte:head>
```

### 3. Open Graph теги (опционально)
Для улучшения отображения в социальных сетях можно добавить:

```html
<meta property="og:title" content="Заголовок страницы" />
<meta property="og:description" content="Описание страницы" />
<meta property="og:type" content="website" />
<meta property="og:url" content="https://bonus5.ru/page-url" />
<meta property="og:image" content="https://bonus5.ru/og-image.jpg" />
```

### 4. Twitter Card (опционально)
```html
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="Заголовок страницы" />
<meta name="twitter:description" content="Описание страницы" />
<meta name="twitter:image" content="https://bonus5.ru/twitter-image.jpg" />
```

## Общие рекомендации

1. **Длина title**: 50-60 символов (оптимально для отображения в поисковой выдаче)
2. **Длина description**: 150-160 символов (оптимально для сниппета)
3. **Keywords**: 5-10 ключевых фраз, разделенных запятыми
4. **Уникальность**: Каждая страница должна иметь уникальные title и description
5. **Релевантность**: Теги должны точно отражать содержимое страницы
6. **Призыв к действию**: В description желательно включать призыв (зарегистрируйтесь, узнайте, начните)
