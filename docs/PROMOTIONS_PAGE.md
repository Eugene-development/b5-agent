# Страница "Акции"

## Описание

Страница "Акции" создана для отображения специальных предложений и акционных программ для агентов платформы BONUS5.

## Расположение

- **Путь к файлу**: `b5-agent/src/routes/(app)/promotions/+page.svelte`
- **URL**: `/promotions`

## Структура страницы

### 1. Hero Section
- Заголовок "Акции"
- Подзаголовок с описанием специальных предложений
- Анимированный индикатор активности

### 2. Акционные предложения (3 карточки)

#### Акция 1: "Завербуй агента"
- **Бонус**: 0.5% с каждой сделки приведенного агента
- **Цветовая схема**: Amber/Orange
- **Преимущества**:
  - Пассивный доход с каждой сделки
  - Неограниченное количество рефералов
  - Выплаты вместе с основным вознаграждением

#### Акция 2: "ТОП 10 агентов"
- **Бонус**: Специальный бонус для лучших агентов 2025 года
- **Цветовая схема**: Purple/Pink
- **Преимущества**:
  - Денежный приз для победителей
  - Статус премиум-агента на 2026 год
  - Эксклюзивные условия сотрудничества

#### Акция 3: "Ноябрьский бонус"
- **Бонус**: +0.2% к каждой сделке до конца ноября
- **Цветовая схема**: Emerald/Teal
- **Срок действия**: До 30.11.2025
- **Преимущества**:
  - Автоматическое начисление бонуса
  - Действует на все новые проекты
  - Ограниченное предложение

### 3. Раздел "Как участвовать"
- Информация об автоматической активации акций
- Пошаговая инструкция (3 шага):
  1. Зарегистрироваться на платформе
  2. Регистрировать клиентов
  3. Получать повышенные бонусы

### 4. CTA Section
- Призыв к действию
- Кнопка регистрации с градиентом amber/orange

## Дизайн

### Цветовая палитра
- **Основной фон**: `bg-gray-950`
- **Акция 1**: Amber/Orange градиенты
- **Акция 2**: Purple/Pink градиенты
- **Акция 3**: Emerald/Teal градиенты
- **Текст**: White/Gray оттенки

### Эффекты
- Hover эффекты с подъемом карточек (`hover:-translate-y-2`)
- Градиентные тени при наведении
- Анимированные индикаторы
- Размытые декоративные элементы

### Адаптивность
- Мобильная версия: 1 колонка
- Планшет и выше: 3 колонки (grid)
- Responsive padding и размеры шрифтов

## Интеграция в навигацию

Ссылка на страницу добавлена в:
1. **Десктопное меню** (`b5-agent/src/routes/layout/header/UI/Menu/index.svelte`)
   - Цвет hover: Amber (`hover:border-amber-400/30`)
   - Активное состояние с amber градиентом
   - Анимированный индикатор снизу

2. **Мобильное меню** (`b5-agent/src/routes/layout/header/UI/MobileMenu/index.svelte`)
   - Кнопка с amber стилизацией
   - Индикатор активной страницы
   - Полная ширина для мобильных устройств

3. **Футер** (`b5-agent/src/routes/layout/footer/index.svelte`)
   - Раздел "Навигация"
   - Hover эффект с amber цветом

### Тематическая цветовая схема
Страница "Акции" использует **Amber/Orange** цветовую схему, которая:
- Привлекает внимание к специальным предложениям
- Создает ощущение срочности и выгоды
- Визуально отличается от других разделов
- Автоматически подсвечивается в меню при активации страницы

Подробнее о тематических цветах меню: `b5-agent/docs/THEMATIC_MENU_COLORS.md`

## Стилистика
