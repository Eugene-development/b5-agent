# Страница "Партнёрство"

## Описание

Создана новая страница `/partnership` в проекте b5-agent, которая предоставляет информацию о партнёрской программе BONUS5 для компаний.

## Расположение

- **Файл страницы**: `b5-agent/src/routes/(app)/partnership/+page.svelte`
- **URL**: `/partnership`

## Целевая аудитория

Страница предназначена для:
- Производителей мебели
- Поставщиков бытовой техники
- Поставщиков мебельной фурнитуры

## Содержание страницы

### 1. Hero секция
- Заголовок с градиентом
- Краткое описание возможностей партнёрства

### 2. О партнёрской программе
- Общая информация о том, как работает платформа для партнёров

### 3. Кто может стать партнёром
- Три категории партнёров с иконками и описанием

### 4. Как работает партнёрство
- Пошаговая инструкция из 5 шагов

### 5. Преимущества для партнёров
- 4 ключевых преимущества:
  - Новые клиенты
  - Оплата за результат
  - Рост продаж
  - Простая интеграция

### 6. Почему выбрать нас
- Качественные лиды
- Прозрачная аналитика

### 7. Условия партнёрства
- Детальное описание условий сотрудничества
- Процент вознаграждения
- Сроки выплат
- Правила повторных сделок

### 8. Call-to-Action
- Кнопка регистрации
- Ссылка на дополнительную информацию

## Навигация

Ссылка на страницу добавлена в:
- **Десктопное меню**: `b5-agent/src/routes/layout/header/UI/Menu/index.svelte`
- **Мобильное меню**: `b5-agent/src/routes/layout/header/UI/MobileMenu/index.svelte`

## Стилистика

Страница выполнена в едином стиле с остальными страницами проекта:
- Тёмная тема (gray-950)
- Градиентные акценты (blue, cyan, teal, emerald)
- Анимации при наведении
- Адаптивный дизайн
- Использование Tailwind CSS
