# Тематическая цветовая схема меню

## Описание

Каждая страница проекта имеет свою уникальную цветовую схему, которая отражается в подсветке соответствующего пункта меню при активации. Это создает визуальную связь между навигацией и содержимым страницы, улучшая пользовательский опыт.

## Цветовая карта страниц

### 1. Главная (`/`)
- **Цвет**: Indigo
- **Hover**: `hover:border-indigo-400/30 hover:bg-indigo-500/15 hover:text-indigo-300`
- **Active**: `border-indigo-400/40 from-indigo-500/25 to-indigo-600/25 text-indigo-400`
- **Индикатор**: `via-indigo-400`

### 2. О проекте (`/about`)
- **Цвет**: Purple
- **Hover**: `hover:border-purple-400/30 hover:bg-purple-500/15 hover:text-purple-300`
- **Active**: `border-purple-400/40 from-purple-500/25 to-purple-600/25 text-purple-400`
- **Индикатор**: `via-purple-400`
- **Обоснование**: Страница использует indigo/purple градиенты в дизайне

### 3. Выплаты (`/payments`)
- **Цвет**: Emerald
- **Hover**: `hover:border-emerald-400/30 hover:bg-emerald-500/15 hover:text-emerald-300`
- **Active**: `border-emerald-400/40 from-emerald-500/25 to-emerald-600/25 text-emerald-400`
- **Индикатор**: `via-emerald-400`
- **Обоснование**: Страница использует emerald/teal градиенты, ассоциируется с деньгами

### 4. Акции (`/promotions`)
- **Цвет**: Amber
- **Hover**: `hover:border-amber-400/30 hover:bg-amber-500/15 hover:text-amber-300`
- **Active**: `border-amber-400/40 from-amber-500/25 to-amber-600/25 text-amber-400`
- **Индикатор**: `via-amber-400`
- **Обоснование**: Страница использует amber/orange градиенты, привлекает внимание к акциям

### 5. Партнёрство (`/partnership`)
- **Цвет**: Cyan (только в футере)
- **Hover**: `hover:text-cyan-400`
- **Обоснование**: Страница использует blue/cyan градиенты

## Реализация

### Десктопное меню
Файл: `b5-agent/src/routes/layout/header/UI/Menu/index.svelte`

Каждая ссылка имеет:
- Базовые стили для неактивного состояния
- Hover эффекты с тематическим цветом
- Active состояние с градиентом и тенью
- Анимированный индикатор снизу при активации

### Мобильное меню
Файл: `b5-agent/src/routes/layout/header/UI/MobileMenu/index.svelte`

Аналогичная логика с адаптацией под мобильный формат:
- Кнопки вместо ссылок
- Полная ширина элементов
- Те же цветовые схемы

### Футер
Файл: `b5-agent/src/routes/layout/footer/index.svelte`

Упрощенная версия с hover эффектами:
- Только изменение цвета текста при наведении
- Соответствие тематическим цветам страниц

## Преимущества подхода

1. **Визуальная связь**: Пользователь сразу понимает, на какой странице находится
2. **Улучшенная навигация**: Цветовое кодирование помогает быстрее ориентироваться
3. **Единообразие**: Цвета меню соответствуют дизайну страниц
4. **Эстетика**: Разнообразие цветов делает интерфейс более живым и интересным
5. **Запоминаемость**: Пользователи ассоциируют разделы с определенными цветами

## Технические детали

### Структура классов
```svelte
class="... hover:border-{color}-400/30 hover:bg-{color}-500/15 hover:text-{color}-300 ... {isActive(path)
  ? 'border-{color}-400/40 from-{color}-500/25 to-{color}-600/25 text-{color}-400 shadow-md shadow-{color}-500/30'
  : ''}"
```

### Индикатор активной страницы
```svelte
{#if isActive(path)}
  <span class="... via-{color}-400 ..."></span>
{/if}
```

## Цветовая палитра Tailwind

Используемые цвета из Tailwind CSS:
- `indigo-400/500/600` - Основной цвет бренда
- `purple-400/500/600` - О проекте
- `emerald-400/500/600` - Выплаты
- `amber-400/500/600` - Акции
- `cyan-400/500/600` - Партнёрство
